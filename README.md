# Digital Autobiography Landing Page

This repository contains the landing page for the Digital Autobiography project.

## 🚀 Get Started (Static Site)

### Option 1: Open Directly (Fastest)

Open `index.html` in your browser.

### Option 2: Run a Local Server (Recommended)

From this folder, run one of the following:

#### Windows / Mac / Linux (Python)

```bash
python -m http.server 5500
```

Then visit:

`http://localhost:5500`

### Option 3: VS Code Live Server

1. Install the Live Server extension in VS Code.
2. Right-click `index.html`.
3. Select **Open with Live Server**.

## Repository Contents

- `index.html` — main landing page

## Important Note

Commands such as:

- `ruby setup_wizard.rb`
- `bundle install`
- `rails server`

are for the separate Ruby on Rails template project, not this landing-page repository.
